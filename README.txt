ReiniciarConexão.bat:	Script que força a renovação de IP em loop, até que o endereço ip iniciado em "10.10" seja obtido pela máquina.

ScriptADP.bat:	Script que permite ao usuário selecionar um dos adaptadores de rede da máquina para ativar ou desativar.

ScriptNET.bat:	Script que permite ao usuário selecionar uma das redes da máquina para forçar a renovação do IP em loop, até que o endereço IP da rede desejada seja obtido pela máquina.
